## Starting
If you want to run this locally, first you should install all dependency needed.
```bash
cd transactions-api
npm i
cd ../transactions-web
npm i
```

For the frontend create `.env` file, and copy content from .env.example
Open two terminals in root.
First one:
```bash
cd transactions-api
npm run start:dev
```

Second one:
```bash
cd transactions-web
npm run dev
```
